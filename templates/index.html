Backend API Server

This is the backend API server for FitFind. The main application frontend is built with Next.js.

For the full experience, please use the Next.js frontend at http://localhost:3000

Open Frontend App API Health Check

API Endpoints

  • POST /api/upload - Upload outfit images
  • POST /api/redo - Redo search with feedback
  • GET /api/results/<file_id> - Get search results
  • GET /api/health - Health check
  • Authentication endpoints: /api/auth/*
  • Wishlist endpoints: /api/wishlist/*
  • History endpoints: /api/history

Development Setup

To start both backend and frontend servers:

python start_dev.py
